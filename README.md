## Installation

1. Clone the repository.
  ```bash
  git clone git@github.com:SuiLotto/suilotto-frontend.git
  ```
2. Install the dependencies.
  ```bash
  npm install
  ```

## Usage

1. Start the development server.
  ```bash
  npm run dev
  ```
